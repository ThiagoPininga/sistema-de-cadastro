Comandos git

  • cd pasta - vai para a pasta desejada no prompt para começar a usar os comandos git
  • clone - clona o repositório
  • commit . -m "mensagem" - commita o novo repositório
  • log (--oneline) - informa as modificações
  • commit - commita o novo repositório
  • push - manda p o repositório do github
  • pull - pega as novas alterações do repositório

Comandos git

  • cd pasta - vai para a pasta desejada no prompt para começar a usar os comandos git
  • clone - clona o repositório
  • commit . -m "mensagem" - commita o novo repositório
  • log (--oneline) - informa as modificações
  • commit - commita o novo repositório
  • status - mostra o status dos arquivos do repositório
  • push (origin nomeDaBranch) - manda p o repositório do github
  • pull - pega as novas alterações do repositório
  • add . - adiciona os novos arquivos ao repositório
  • restore --source codigoDoCommit . - restaura todos os arquivos até certo commit que foi indicado
  • checkout -b nomeDaBranch - cria nova branch
  • switch nomeDaBranch - troca a branch para a desejada
  • branch - mostra quais branch's existem e qual você está agora
  • merge nomeDaBranch - juntar as branch's (com o nome da que vai passar para a main)
